# Menu Selection SPA

Две страницы на Angular 19: приветствие и меню с чекбоксами. В шапке меню — раздел, число выбранных пунктов и сумма значений.

## Demo

- https://willalone.github.io/menu-selection-spa/
- https://willalone.github.io/menu-selection-spa/#/menu

На GitHub Pages маршруты через hash (`#/menu`). Локально — `/` и `/menu`.

## Запуск

```bash
npm install
npm start
```

http://localhost:4200

## Сборка

```bash
npm run build
npm run build:gh-pages   # для GitHub Pages
```

## Тесты

```bash
npm test
```

Сервис состояния, чекбоксы, роутинг.
